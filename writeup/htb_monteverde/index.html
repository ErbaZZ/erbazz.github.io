[Writeup] Hack The Box :: Monteverde

Monteverde is a Medium difficulty Windows box. You don’t need many fancy techniques to clear this box, as the process is quite clear and straightforward. The important point for this box is enumeration, as the saying goes, “Enumeration is the key”.

OS: Windows
Difficulty: Medium
Points: 30
Release: 11 Jan 2020
IP: 10.10.10.172
Date Cleared: 23 Apr 2020

This Writeup is Encrypted

Writeups for active machines are only accessible to people who have cleared them.

Unlocking

Linux Machine: Use the Root password hash from /etc/shadow (copy from root to :::)
For example: root:$6$BGk6CBPE$FoDCUgY.1p——redacted——cw/D/3ubU/:18222:0:99999:7:::

Windows Machine: Use the Administrator SAM hash (copy from Administrator to :::)
For example: Administrator:500:aad3b435b51404eeaa——redacted——80c4fd1448319a8c04f:::
