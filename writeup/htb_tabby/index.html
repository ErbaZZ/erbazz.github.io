[Writeup] Hack The Box :: Tabby

Here’s my writeup for Tabby, a Linux box on Hack The Box. This one is created by egre55 and it is rated as Easy. The techniques involed to clear Tabby are not complicated, but you need to connect the dots, using information/vulnerabilities from one component to gain access to other components.

OS: Linux
Difficulty: Easy
Points: 20
Release: 20 Jun 2020
IP: 10.10.10.194
Box Creator: egre55
Date Cleared: 5 Aug 2020

This Writeup is Encrypted

Writeups for active machines are only accessible to people who have cleared them.

Unlocking

Windows Machine: Use the Administrator SAM hash (copy from Administrator to :::)
For example: Administrator:500:aad3b435b51404eeaa——redacted——80c4fd1448319a8c04f:::
